Installation
============
فایل را از مخزن دانلود کنید

یا میتوانید دستور زیر را وارد کنید:
::
  git clone https://github.com/M3HR4NUS/zawa.git
  cd zawa
  
  
بعد دستور زیر را وارد کنید:
::
  npm i
  

بعد دستور زیر را وارد کنید:
::
  node app
  
==============

الان پورت 3000 درحال اجراس:
::
  http://localhost:3000/home
